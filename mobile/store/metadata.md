## Description (long)

Melange is where photographers, models, makeup artists, stylists, and designers find each other.

Post a project you're working on. Swipe through other creatives' posts. Match when you both like each other's work. Chat in realtime to plan the shoot.

**What's inside:**
- Profile with avatar, role, skills, bio, and current project
- Post a collab — title, description, location, compensation, tags, multiple photos
- Swipe deck of nearby creatives with search filters
- Real-time chat for matches with read receipts
- Push notifications for new matches and messages
- Block and report tools to keep the community safe

Free to use. No ads, no in-app purchases.
